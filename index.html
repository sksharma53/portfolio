Capstone Project

The focal point of my capstone project is to enhance efficiency and productivity at Acxiom by implementing a system to track weekly working hours and discern patterns in employee schedules. The primary objective is to identify which days exhibit higher work engagement across the organization. To achieve this, the project will employ a systematic analysis and design approach.

The proposed system will integrate with existing data sources, such as timesheets and attendance records, to gather information on employee working hours. Through data analysis, the system will pinpoint peak working days and propose adaptive strategies. Potential risks include inaccuracies in data analysis and the need to carefully balance tracking efforts to avoid negatively impacting the employee experience.

The stakeholders for this project include employees, managers, and human resources personnel. The system's architecture will involve integrating with relevant data sources and implementing features to adapt the tracking process. The user interface will prioritize user experience, providing intuitive navigation for users to access their working hour details easily.

The database design will focus on collecting and analyzing data on working hours to inform adaptive strategies. Rigorous testing, including data analysis, user testing, and system integration testing, will be conducted to ensure the system meets requirements. The implementation plan will be phased, addressing specific aspects of the tracking process, and a maintenance plan will ensure continuous improvement and alignment with organizational needs.

Through this structured approach, the project aims to enhance organizational insight into employee work patterns and contribute to optimizing productivity at Acxiom.
